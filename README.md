Pose Classification & Bodybuilding Guide

This project uses MediaPipe and OpenCV to detect, classify, and provide feedback on bodybuilding and yoga poses in real-time. It can also display reference images alongside the live camera feed, giving visual guidance for correct posture.

Features

Real-time pose detection using your webcam.

Pose classification for classic bodybuilding poses:

Front Lat Spread

Front Double Bicep

Side Tri

Rear Lat Spread

Back Double Bicep

Side Chest

Most Muscular

Abs & Thighs

Angle-based scoring system to evaluate how closely your pose matches the target pose.

Tips and feedback displayed directly on the video feed.

Reference images overlay to guide correct posture.
